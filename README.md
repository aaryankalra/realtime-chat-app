### npm install in the root folder
```
npm i
```
### npm install in the frontend folder
```
cd frontend
npm i
```
### Create .env file in the root folder
```
PORT =
MONGO_DB_URI =
JWT_SECRET =
NODE_ENV =
```
### Run the server
```
npm run server
```
### Start the frontend
```
cd frontend
npm run dev
```
